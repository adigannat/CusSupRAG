# CusSupRAG

This is a Customer Support RAG inteded for developers to have their questions about AWS resources quickly and accurately answered.
As the depth of the AWS suite of services is far too great for any one developer to keep track of easily,
this RAG is intended to facilitate ease of development by having the correct answers to questions about AWS services quickly and accurately answered.
